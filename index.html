El objetivo de este proyecto es desarrollar una plataforma web que combine la experiencia visual característica de aplicaciones como Instagram con la comunicación rápida y dinámica que ofrece X. Además, la plataforma contará con dos secciones diferenciadas: una destinada a publicaciones de un tipo específico y otra para publicaciones de distinta naturaleza. Los usuarios podrán interactuar no solo a través de comentarios, sino también mediante un sistema de mensajería directa, fomentando una comunicación más completa y eficiente.

PHP JavaScript HTML CSS

El objetivo de este proyecto consiste en una plataforma web similar a Wallapop. En ella, los usuarios pueden publicar los productos que desean vender y realizar ofertas por aquellos disponibles. Además, en el proceso de negociación, comprador y vendedor tienen la posibilidad de intercambiar una serie de contraofertas hasta alcanzar un acuerdo. Este desarrollo se llevó a cabo como parte del aprendizaje de PHP en el programa de DAW y permitió consolidar conocimientos sobre la lógica de negocio en aplicaciones web dinámicas.
